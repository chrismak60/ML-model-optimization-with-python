This project aims at showcasing model optimization techniques on Machine Learning models. 
Demonstration of these procedures was done on two datasets and objectives:
 - One on a USA housing dataset, predicting housing prices
 - Second one on a Telecom Churn dataset, predicting Churn

The techinques shown consist of using VIF to find multicolliniarity to reduce redundancy, 
WOE (weight of evidence ) and IV (invofmation values) to determine feature importance
and PCA for dimensionality reduction on multi-dimensional feature sets.
